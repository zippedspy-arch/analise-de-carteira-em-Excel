# Otimização de Portfólio em Excel – Modelo de Markowitz

## Visão Geral

Este projeto aplica a teoria de portfólio de média-variância de Markowitz utilizando Excel.

A análise foi desenvolvida com base em dados históricos mensais dos ativos PETR4 e BOVA11, com o objetivo de avaliar métricas de risco e retorno e identificar a carteira de mínima variância.

## Etapas da Análise

- Organização dos preços históricos dos ativos
- Cálculo dos retornos mensais
- Cálculo da volatilidade
- Análise de covariância e correlação entre os ativos
- Construção da fronteira eficiente
- Identificação da carteira de mínima variância
- Análise da composição da carteira ótima

## Resultados

A carteira de mínima variância encontrada foi composta por:

- **10% PETR4**
- **90% BOVA11**

Essa carteira apresentou volatilidade menor do que os ativos individuais, evidenciando o benefício da diversificação entre os ativos.
